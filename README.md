# Iris Classification Project

## Objective

Build a machine learning model to classify Iris flower species using sepal length, sepal width, petal length, and petal width.

## Dataset

Iris Classification Dataset

## Algorithms Used

* K-Nearest Neighbors (KNN)
* Logistic Regression
* Decision Tree

## Results

| Model               | Accuracy |
| ------------------- | -------- |
| KNN                 | 100%     |
| Logistic Regression | 96.67%   |
| Decision Tree       | 93.33%   |

### Best Model

KNN (100% Accuracy)

## Sample Prediction

Input:

* Sepal Length: 5.1
* Sepal Width: 3.5
* Petal Length: 1.4
* Petal Width: 0.2

Output:
Iris-setosa

## Conclusion

The KNN model achieved the highest accuracy of 100% and was selected as the final model for deployment and prediction.
